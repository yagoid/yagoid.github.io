    Pryecto destacado

    Meraki

    Meraki es un aplicación para Android cuyo objetivo principal es permitir a las tiendas más pequeñas la opción de llegar a un mayor número de clientes. Gracias a Meraki, los usuarios que entren en ella, podrán observar las tiendas que se han dado de alta en su zona, de este modo, podrás ver su localización exacta, los productos existentes, y mucha más información.

    • Antroid Studio
    • Firebase
    • Java
    • API Maps
    • Google Play
